<h1 align="center">🏏 IPL 2026 Auction Web Scraping Project</h1>

<p>
This project is a Python-based web scraping application that extracts sold player details from the official IPL Auction website. Using BeautifulSoup and Requests, it collects information such as player names, teams, sold prices, base prices, and bidding details. The scraped data is cleaned, transformed using Pandas, and exported into a structured dataset for further analysis and reporting.
</p>

<h2>✨ Features</h2>

<ul>
  <li>Scrapes IPL 2026 Auction sold player information from the official IPL website.</li>
  <li>Extracts player names, teams, sold prices, base prices, and bidding details.</li>
  <li>Handles HTTP requests, response validation, and exception handling.</li>
  <li>Cleans and transforms scraped data using Pandas.</li>
  <li>Exports structured data for analysis and reporting.</li>
</ul>

<h2>🛠️ Tech Stack</h2>

<ul>
  <li>Python</li>
  <li>Requests</li>
  <li>BeautifulSoup (bs4)</li>
  <li>Pandas</li>
</ul>
